# data_structure_kruskal
A homework for year1 data_structure learning - kruskal

## 实验目的：
掌握求最小生成树的两种方法，并进行程序实现。

## 实验内容：
1. 实验内容：
给定一个地区的n个城市间的距离网，用Prim算法或Kruskal算法生成最小生成树，并计算得到的最小生成树的代价。要求显示得到的最小生成树中包括了哪些城市间的道路，并显示得到的最小生成树的代价。
2. 基本要求:
从键盘输入n个顶点和m条边(6≤n≤15，n-1≤m≤20)，建立图的邻接表(邻接矩阵也可)存储图，然后输出该邻接表(邻接矩阵)，用Prim或Kruskal算法求出其最小生成树，输出最小生成树中的城市、城市间的道路及距离和最小生成树的代价。输入输出的形式可参见“五、测试与输出结果”。
